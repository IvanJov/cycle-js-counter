# cycle-js-counter

This is the code for tutorial posted on sitepoint.

## Running the app
#### Clone the code 
```
git clone https://github.com/IvanJov/cycle-js-counter.git && cd cycle-js-counter
```

#### Run app
```
npm start
```

#### Open `index.html` in browser

You should see something like this:
